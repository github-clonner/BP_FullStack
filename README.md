App Boilerplate for myself.

# What's inside

* Universal(isomorphic) Render
* React/Redux/Node/Express/MongoDB/Passport/JWT
* React-Router 4
* Redux-Form 6.8
* Authentication(Signin, Signup, Forgot, Reset)
* Mailgun
* Flow
